## 배포
1. GitHub에서 Public 저장소 `kira-prompt.github.io` 생성
2. ZIP 내부 파일 전체를 저장소 루트에 업로드
3. Settings → Pages → Deploy from a branch → main / root

## 주의
페이지 구조와 복사 기능은 완성되어 있습니다. 노션에서 전체 프롬프트를 안전하게 추출하지 못한 칸은 `[PROMPT NOT INSERTED]`로 남겨 두었습니다. 각 경로의 index.html 내부 `window.PAGE_DATA` 안 prompt 값을 실제 프롬프트로 교체하세요.
